# Week-4-Project-02 Registration Form Using Textarea
Registration Form with Character Counter

A simple and responsive Registration Form built using HTML, CSS, and JavaScript.
This project demonstrates form input handling, textarea character counting, and basic front-end validation.

Features

User-friendly registration form

Responsive UI design using CSS

Textarea with live character counter

Basic form validation using JavaScript

Clean and simple layout for beginners

Technologies Used

HTML5 – Structure of the form

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Character counter and validation

Project Structure
registration-form/
│
├── index.html
├── style.css
├── script.js
└── README.md
How It Works

User enters details such as name, email, and message.

The textarea includes a character limit.

JavaScript dynamically displays the remaining characters while typing.

Basic validation ensures required fields are filled before submission.
